一，框架对比

  • Vue：轻量级，快速开发，较小工程
  • React：所有都是Js，依赖各种开源类库
  • Angular：大型项目，多人合作开发，开箱即用

二，安装

  1. npm i -g @angular/cli
  2. ng version
  3. ng new projectName
  4. ng serve

三，项目目录

  • 版本号：~：锁定版本号前两位、^：锁定版本号前一位
